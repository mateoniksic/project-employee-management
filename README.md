# Employee Management

## Tech Stack
- ReactJS *(18.2.0)*
- PostgreSQL (Supabase)
- HTML / CSS
- TypeScript

## How to start the application?

1. **Clone repository**
2. **Run command:** `npm run dev`
3. **Open in browser:** http://localhost:5173/
4. **Stop command:** CTRL + C
